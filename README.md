# 🎯 AI Career Advisor – Gemini Powered GenAI Chatbot

A production-ready, domain-specific Career Advisor chatbot built using **Google Gemini API** and **Streamlit Cloud**, designed to provide structured and personalized career guidance.

This project demonstrates real-world **Generative AI engineering practices**, including modular architecture, advanced prompt engineering, multi-turn conversation memory, and secure API handling.

---

## 🚀 Features

- Structured Career Insights
- Skill Gap Analysis
- Personalized Learning Roadmap
- Resume Improvement Tips
- Interview Preparation Guidance
- Session-Based Multi-Turn Memory
- Advanced Prompt Engineering
- Secure API Key Management (.env)
- Professional Chat-Style UI
- Streamlit Cloud Deployment

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/ai-career-advisor-gemini.git
cd ai-career-advisor-gemini
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Create `.env` File

Create a `.env` file in the root directory:

```
gemini_key=YOUR_GEMINI_API_KEY
```

### 4️⃣ Run Application

```bash
streamlit run app.py
```
